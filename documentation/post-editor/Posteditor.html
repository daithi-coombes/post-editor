Class Details

	
This class loads jquery-ui themes/plugins and tinymce custom plugin



The tinymce plugin can be found at:  POSTEDITOR_DIR . "/application/includes/tinymce..../posteditormodal/".  The plugin opens a dialog with an iframe pointing to wp-admin/admin-ajax.php.  The ajax call loads the view file for the PosteditorModal module.
For adding more buttons and actions to the dialog see the description in  PosteditorModal.
